Welcome to the University of Western Australia Biomedical School Biological databse. This utility is provided to aid in recording sample locations within the freezer and Liquid Nitrogen facilities of L block at the QEII Medical centre. How to use the system: Access storage locations by viewing the rooms or freezers page. You will be able to follow navigation links to relative boxes contained within each of the facilities. Each box page will display a table of box locations and any relevant samples contained within. Clicking on a box location will reveal further details about the sample contained within. You may create additional samples on box pages using the create link. These will be added to the box location specified. remove samples from box locations if you have removed them from that location.
